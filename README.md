## Description

User Mobile Application for the Courier Services Domain

## Overview

This repository is part of the Social E-commerce Ecosystem microservices architecture, focusing on the Courier Services domain.

## Tech Stack Recommendations

- **Backend**: Node.js/Express, Java/Spring Boot, or Python/FastAPI
- **Frontend**: React/React Native for web and mobile applications
- **Database**: PostgreSQL, MongoDB, Redis, Elasticsearch
- **Message Broker**: Kafka or RabbitMQ
- **Containerization**: Docker and Kubernetes
